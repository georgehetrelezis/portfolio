Royal Wolf Website Homepage

This collaborative project was for Royal Wolf's 2021 website rebuild, as I was assigned the website's home page with the support of the team's Senior Developers at Milestone Belanova.

Project was created with a custom-built CMS in the PHP framework, Laravel.

Stadium Sports Floors Website

My first solo project in a professional environment, this website was developed for Stadium Sports Floors based in South Australia. Project was developed while at Milestone Belanova.

Project was created with a custom-built CMS in the PHP framework, Laravel.

AppSync Angular Proof of Concept

Proof of concept app which allowed multiple devices in educational spaces to be in sync when AV sources and volume levels are updated. Developed during my internship at Innomate.

Project was created with Angular, AWS AppSync, and C# .NET.

Doorbell Proof of Concept

This Proof of Concept function was built on top of an existing system to allow visitors to notify staff when they've visited an office. Project was worked on during my internship at Innomate.

Feature was built in C# .NET and Crestron UI Tools.

Central Coast Haematology Website

This website was built for the Central Coast Haematology departments located in Kanwal and North Gosford, as I was responsible for the project until it was completed by another member of the Milestone Belanova team.

Project was created with a custom-built CMS in the PHP framework, Laravel.
